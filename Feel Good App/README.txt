////////////////////////FEEL GOOD APP///////////////////////////

Highlights -:

1. This app allows users to login, or signup if the user is new, and then it takes the user's mood as input and displays quotes accordingly.

2. The app is build solely on Python using kivy, json, hoverable, pathlib and datetime. It uses classes to define different screens and then connects them all using main.py

3. It stores username and password information in a separate file named users.json . 

4. Special feature includes hoverable and interactive 'LOGOUT' button.
